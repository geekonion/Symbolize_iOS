# Symbolize_iOS

## Installation

1. Clone this repo

2. execute setup.sh

   ```stylus
   $ ./setup.sh 
   $ source ~/.bash_profile
   ```

## Usage

```stylus
$ symbolize /Users/xxx/test/JITDemo-2024-08-20-215613.ips 
--共有1个image需要符号化
--开始查找符号文件
--结束查找符号文件
--开始符号化
符号化 JITDemo 5个符号
符号化的崩溃日志已写入/Users/xxx/test/JITDemo-2024-08-20-215613.sym.crash
```
